Spaghetti

Description

This is a finished plate of spaghetti using a bolognese sauce.

It is made using canned tomato sauce and store bought meatballs. The finished product is fantastic!

Ingredients

  • Tomato sauce
  • Meatballs
  • Cheese
  • Spaghetti noodles

Steps

  1. Boil water for noodles then add noodles.
  2. While noodles are cooking start heating up tomato sauce
  3. Start meatballs in oven
  4. When everything is finished cooking then add all together..
